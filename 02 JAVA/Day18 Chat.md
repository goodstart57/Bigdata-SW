## 2. Thread 연습

Coding
```java
public class Main {
    public static void main(String[] args) {
        A t1=new A();
        t1.start();
        A t2=new A();
        t2.start();
        
    }
}

class A extends Thread {
    public void run() {
        for(int i=0; i<5; i++) {
           System.out.println(i+"+1="+(i+1));
        }   
    }
}
```

Output
```
0+1=1
1+1=2
2+1=3
3+1=4
4+1=5
0+1=1
1+1=2
2+1=3
3+1=4
4+1=5
```

쓰레드는 순차적으로 실행되는게 아니라 독립적으로 실행된다.

start(), stop(), sleep() 등을 통해서 쓰레드의 실행 흐름을 제어할수있다.

```
start()            : 쓰레드를 실행한다.

stop()             : 쓰레드를 비정상 종료한다. 안전하지 않은 메소드

sleep(long millis) : millis 밀리초만큼 쓰레드를 잠재운다.(대기)
```

## 3. Socket 연결하기(Server)

```java
public class ChatServer {
    ArrayList<DataOutputStream> list=new ArrayList<DataOutputStream>();
    public void broadcast(String msg) {   
        synchronized(list) {
            for(DataOutputStream out:list) {
                try {
                    out.writeUTF(msg);
                } catch (IOException ex) {
                    list.remove(out);
                }
            }
        }
    }
    public ChatServer() {
        try {
            ServerSocket ss=new ServerSocket(7777); //1025~66535
            System.out.println("server ready");
            while(true) {
                Socket s=ss.accept();
                System.out.println("연결성공");
                DataInputStream in=new DataInputStream(s.getInputStream());
                DataOutputStream out=new DataOutputStream(s.getOutputStream());
                ServerThread thread=new ServerThread(s, in, out);
                thread.start();
            }
        } catch (IOException ex) {
            Logger.getLogger(ChatServer.class.getName()).log(Level.SEVERE, null, ex);
        }
    }
            
    public static void main(String[] args) {
        new ChatServer();
    }
    
    class ServerThread extends Thread {
        Socket s;
        DataInputStream in;
        DataOutputStream out;
        private ServerThread(Socket s, DataInputStream in, DataOutputStream out) {
            this.s=s;
            this.in=in;
            this.out=out;
        }

        @Override
        public void run() {
            try {
                while(true) {
                    //클라이언트가 보낸 메세지 읽기
                    String msg=in.readUTF();
                }
            } catch (IOException ex) {
                try {
                    System.out.println(s.getInetAddress()+"님 퇴장");
                    if(in!=null) in.close();
                    if(out!=null) out.close();
                    if(s!=null) s.close();
                } catch (IOException ex1) {
                    Logger.getLogger(ChatServer.class.getName()).log(Level.SEVERE, null, ex1);
                }
            }
        }
        
    }
}
```

ServerSocket 클래스를 이용하여 소켓 생성후 연결한다.
